# Ansible SmartOS Server Configuration

Deploys a number of services into joyent-brand zones on a SmartOS server

Current roles:

- Bare metal (deploys zones)
- Gateway (manages firewalling and routing)
- Rsyncd
- Rsyslog
- M1cr0blog
- InfluxDB
- Chronograf
- Telegraf
- Traefik

Dependencies:

- None!
